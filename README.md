## Project Overview

96 RANCH Restaurant LP is a modern restaurant landing page designed to transform a traditional restaurant website into an engaging food discovery experience.

The project combines:

* Food storytelling
* Mobile-first design
* Netflix-inspired food browsing
* Social media integration
* Conversion-focused user experience
